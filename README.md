# Student Management System

## Overview
This is a DBMS-based Student Management System developed using PHP, MySQL, HTML, CSS, and XAMPP. The application helps manage students, teachers, courses, departments, and enrollments efficiently.

## Features
- Student Management
- Teacher Management
- Course Management
- Department Management
- Enrollment Management
- CRUD Operations (Create, Read, Update, Delete)

## Technologies Used
- PHP
- MySQL
- HTML5
- CSS3
- XAMPP

## Database
Import the `student_db.sql` file into phpMyAdmin before running the project.

## How to Run
1. Install XAMPP.
2. Copy the project folder into `htdocs`.
3. Start Apache and MySQL.
4. Import `student_db.sql` into phpMyAdmin.
5. Open:
   ```
   http://localhost/student-management-system/
   ```
